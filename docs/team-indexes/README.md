# 팀별 참조 문서 인덱스

> 목적: 팀별 기능, 핵심 스크립트, 구조, 운영 설정, 관련 문서를 한 장씩 빠르게 찾기 위한 인덱스다.

## 문서 읽는 법

- 팀 문서는 가능하면 같은 순서로 읽는다.
  - `역할`
  - `핵심 기능`
  - `핵심 진입점`
  - `운영 스크립트`
  - `운영 설정`
  - `자주 쓰는 명령어`
  - `관련 문서`
- 급하면 `자주 쓰는 명령어`와 `운영 설정`만 먼저 봐도 된다.
- 팀 문서에 새 기능을 추가할 때는 “코드 위치를 바로 찾을 수 있는지”를 기준으로 링크를 보강한다.
