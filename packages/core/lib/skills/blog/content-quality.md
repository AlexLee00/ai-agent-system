# Blog Skill: Content Quality

목표:
- 기계적 문체를 줄이고 실제 사람의 글처럼 리듬과 밀도를 만든다.
- 품질 검증 기준을 글쓰기 단계에서 먼저 반영한다.

필수 규칙:
- 문단 길이를 일부러 다르게 가져간다.
- 같은 문장 패턴을 3번 이상 반복하지 않는다.
- 섹션마다 한 줄 요약 또는 전환 문장을 넣는다.
- 구체 명사, 실제 상황, 실제 판단 기준을 우선한다.
- 코드/체크리스트/예시 중 최소 1개는 반드시 포함한다.

검증 포인트:
- 최소 글자 수 충족
- 섹션 누락 없음
- 질문형 FAQ 포함
- 카페/운영 홍보 문구의 명칭이 정확함
- 미래 글/없는 글 내부링크 금지
