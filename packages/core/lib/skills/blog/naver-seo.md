# Blog Skill: Naver SEO

목표:
- 네이버 검색과 AI Briefing이 글 구조를 빠르게 이해할 수 있게 만든다.
- 핵심 요약, 질문형 섹션, 구조화된 본문을 우선한다.

필수 규칙:
- 서론 직후 `[핵심 요약 3줄]` 또는 `[AI 스니펫 요약]`을 둔다.
- `[이 글에서 배울 수 있는 것]`을 불릿 3~5개로 제시한다.
- 질문형 Q&A를 3개 이상 포함한다.
- 결론에는 “핵심 메시지 한 줄”이 드러나야 한다.

권장 구조:
1. 요약
2. 독자 문제 정의
3. 구조화된 본문
4. 실전 사례
5. Q&A
6. 결론
7. 해시태그

금지:
- 추상적인 키워드 반복
- 본문보다 긴 인사말
- 결론 없이 끝나는 구성
