# Model Selection

Use selector keys, not provider strings, as the operational control point.

- Bulk or summary work: haiku, mini, or fast chains.
- Long-form synthesis: sonnet candidate under a team gate.
- High-risk judgment: opus or code-capable route with explicit selector.
- Embedding: local embedding only, not chat fallback.

If `LLM_AUTO_ROUTING_ENABLED=shadow`, compare the policy output with the current selector but do not treat it as active.
