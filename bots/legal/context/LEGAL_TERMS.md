# 법률 용어 사전 — SW 감정 전문 용어

## 기본 법률 용어

| 용어 | 정의 |
|------|------|
| 감정 촉탁 | 법원이 전문가에게 특정 사항의 조사/판단을 의뢰하는 것 |
| 감정인 | 법원의 촉탁을 받아 감정을 수행하는 전문가 |
| 감정서 | 감정인이 감정 결과를 기술하여 법원에 제출하는 문서 |
| 원고 | 소송을 제기한 쪽 |
| 피고 | 소송을 당한 쪽 |
| 감정사항 | 법원이 감정인에게 판단을 요청한 구체적인 질문 사항 |
| 석명 | 당사자나 감정인에게 불명확한 사항을 명확히 설명하도록 요구하는 것 |
| 보충 감정 | 최초 감정 후 추가 확인이 필요한 사항에 대한 재감정 |

## 저작권 관련

| 용어 | 정의 |
|------|------|
| 저작권 침해 | 타인의 저작물을 무단으로 복제/배포/수정하는 행위 |
| 실질적 유사성 | 저작물의 창작적 표현이 실질적으로 유사한 정도 |
| 아이디어/표현 이분법 | 아이디어 자체는 보호 안 되고, 그 구체적 표현만 보호됨 |
| 창작성 | 저작물이 저작권 보호를 받기 위한 최소한의 독창성 |
| 접근 가능성 | 피고가 원고의 저작물에 접근할 수 있었을 가능성 |
| 복제권 | 저작물을 유형물에 고정하거나 유형물로 다시 제작할 권리 |
| 2차적 저작물 | 원저작물을 번역/편곡/변형 등으로 작성한 저작물 |

## 소프트웨어 하자 관련

| 용어 | 정의 |
|------|------|
| 하자 | 계약에서 정한 품질이나 기능을 갖추지 못한 결함 |
| 하자담보책임 | 매도인/수급인이 하자에 대해 지는 법적 책임 |
| 수정 요구권 | 하자가 있는 경우 그 보수를 요청할 수 있는 권리 |
| 기성금 | 공사/개발 진행에 따라 단계별로 지급하는 공사대금 |
| 지체상금 | 납기 지연 시 계약에서 정한 손해배상액 |
| 검수 | 납품물이 계약 조건을 충족하는지 확인하는 절차 |

## 손해배상 관련

| 용어 | 정의 |
|------|------|
| 통상손해 | 채무불이행으로 통상 발생하는 손해 |
| 특별손해 | 특별한 사정으로 인한 손해 (당사자가 알았을 때만 배상) |
| 일실이익 | 손해가 없었다면 얻을 수 있었던 이익 |
| 위자료 | 정신적 손해에 대한 금전적 배상 |

## 코드 분석 관련

| 용어 | 정의 |
|------|------|
| 소스코드 유사도 | 두 코드 간 일치 정도를 수치화한 것 |
| AST | Abstract Syntax Tree — 코드의 문법 구조를 트리로 표현 |
| 난독화 | 코드의 의미를 유지하면서 읽기 어렵게 만드는 기술 |
| 리팩토링 | 기능은 유지하면서 코드 구조를 개선하는 것 |
| 코드 클론 | 동일하거나 유사한 코드 조각이 반복되는 것 |
| 오픈소스 | 소스코드가 공개된 소프트웨어 |
| API | Application Programming Interface — 프로그램 간 통신 규약 |
| DB 스키마 | 데이터베이스의 구조와 제약 조건 정의 |
