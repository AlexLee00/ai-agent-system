# Diagnosis Reading

- Read lowest crank axis first, then title-pattern concentration.
- Treat writer-model comparison as inconclusive when sample is below 5.
- `metadata.writer_model` is an evidence tag for weekly comparison, not a live promotion.
- If all rows are haiku, the report is a baseline, not a winner.
- Use observations to choose the next master-approved `BLOG_WRITER_MODEL` window.
- Do not change YAML, plist, launchd, or writer env from a diagnosis alone.
