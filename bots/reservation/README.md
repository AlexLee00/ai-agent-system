# 📅 예약관리봇

네이버 스마트플레이스 + 픽코 예약 자동화 봇

## 진행 현황 (~85%)

| 기능 | 상태 |
|------|------|
| 로그인 자동화 | ✅ 완료 |
| 네이버 스마트플레이스 연동 | ✅ 완료 |
| 픽코 관리자 페이지 연동 | ✅ 완료 |
| 캘린더 → 홈화면 복귀 자동화 | 🔄 수정 중 |
| 10분 간격 자동 모니터링 | ⏳ 대기 |
| 텔레그램 알림 연동 | ⏳ 대기 |

## 알려진 이슈

```
문제: 캘린더 URL 감지 후 홈화면 이동 텍스트 클릭 실패
해결: 홈 URL 직접 이동 방식으로 수정
     → https://partner.booking.naver.com/bizes/{ID}/home
```

## 파일 설명

| 파일 | 설명 |
|------|------|
| `naver-monitor.js` | 네이버 스마트플레이스 모니터링 (메인) |
| `pickko-final.js` | 픽코 최종본 |
| `pickko-accurate.js` | 픽코 정확도 개선본 |
| `pickko-booking-v2.js` | 픽코 예약 v2 |
| `pickko-auto-register.js` | 픽코 자동 등록 |

## 실행

```bash
npm run monitor  # 네이버 모니터링 시작
npm start        # 봇 전체 실행
```
