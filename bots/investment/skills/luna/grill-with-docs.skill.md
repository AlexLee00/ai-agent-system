# Grill With Docs

- Owner: luna
- Category: meeting-room
- Contract: Challenge a boundary or master-grade Luna meeting decision with five questions and cite only supplied evidence.
- Safety: advisory/shadow only; do not apply trades, parameters, launchd, or runtime changes.

## Required Questions
1. What is the strongest counterargument against this decision?
2. Which plan-note, OOS, parameter-store, or registry evidence would invalidate this decision?
3. What should the master ask before accepting this decision?
4. What is the urgency, and what is lost if this waits?
5. What happened in similar past ADRs or shadow outcomes?

Do not invent citations. Cite only evidence passed in the input. If evidence is unavailable, say `근거 부족`.
