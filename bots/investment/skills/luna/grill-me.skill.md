# Grill Me

- Owner: luna
- Category: meeting-room
- Contract: Challenge one Luna meeting-room decision with five concise adversarial questions.
- Safety: advisory/shadow only; do not apply trades, parameters, launchd, or runtime changes.

## Required Questions
1. What is the strongest counterargument against this decision?
2. Which observed data would invalidate this decision?
3. What should the master ask before accepting this decision?
4. What is lost if this is not decided now?
5. How did similar past decisions perform?

Use only the supplied agenda, plan-note, and evidence. If evidence is missing, say `근거 부족`.
