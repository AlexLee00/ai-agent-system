# learning-skill

## Purpose
Learning implements the Hermes 4-stage loopback for Claude-team operational patterns.

## Stages
1. Observe: collect evidence from repeated incidents, reviews, or successful fixes.
2. Extract: convert repeated evidence into a stable pattern.
3. Promote: propose a skill/check/gate candidate.
4. Verify: require quality-gate validation before active use.

## Outputs
- Pattern key and confidence.
- Skill or gate candidate.
- Verification requirements.

## Safety
Learning proposes changes. It does not auto-write production skills, restart services, or alter protected runtime state.
