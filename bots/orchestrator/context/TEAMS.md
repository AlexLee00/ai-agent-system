## 📅 스카팀 — 스터디카페 운영 관리

**역할**: 스터디카페 예약 현황, 매출 통계, 모니터링 봇 관리

### 팀원

| 봇 | 역할 |
|----|------|
| **스카** | 팀장, 명령 처리 |
| **앤디** | 네이버 스마트플레이스 모니터링 (5분 주기) |
| **지미** | 픽코 키오스크 예약 모니터링 |
| **레베카** | 매출 예측 분석 |
| **이브** | 공공API 수집 (공휴일·날씨·학사·축제) |

### 지원 명령어 (bot_commands)

| command | 설명 | 자연어 예시 |
|---------|------|------------|
| `query_reservations` | 오늘 예약 목록 조회 | "오늘 예약 뭐 있어", "예약 현황 알려줘", "오늘 예약 몇 건이야", "예약 확인해줘" |
| `query_today_stats` | 오늘 매출·입장수 조회 | "오늘 매출 얼마야", "오늘 얼마 벌었어", "오늘 손님 몇 명", "오늘 통계 줘", "수익 확인" |
| `query_alerts` | 미해결 알람 조회 | "알람 뭐 있어", "미해결 있어?", "문제 있어?", "이상한 거 있어?", "경고 확인" |
| `restart_andy` | 앤디 재시작 | "앤디 재시작해", "앤디 다시 시작해", "앤디 죽었어", "앤디 안 돼" |
| `restart_jimmy` | 지미 재시작 | "지미 재시작해", "지미 다시 시작해", "지미 죽었어", "지미 안 돼" |

## 🌙 루나팀 — 암호화폐·주식 자동매매

**역할**: 바이낸스(BTC/ETH/SOL/BNB) 크립토 자동매매, KIS 국내/해외 주식 관리

### 팀원

| 봇 | 역할 |
|----|------|
| **루나** | 팀장, 최종 매수/매도 판단 (gpt-4o) |
| **오라클** | 온체인·파생 데이터 분석 (gpt-4o) |
| **네메시스** | 리스크 평가 — APPROVE/ADJUST/REJECT (gpt-4o) |
| **아테나** | 매도 관점 근거 + 손절가 제시 (gpt-4o) |
| **제우스** | 매수 관점 근거 + 목표가 제시 (gpt-4o) |
| **헤르메스** | 뉴스 수집 + 감성 분류 (Groq) |
| **소피아** | 커뮤니티 감성 분석 (Groq) |
| **아르고스** | Reddit 전략 추천 수집 (Groq) |
| **헤파이스토스** | 자동화·실행 |
| **한울** | 국내 주식 담당 |

### 운영 상태

- **Phase 3-A 크립토**: ✅ OPS (PAPER_MODE=false, 바이낸스 실거래, 30분 사이클)
- **Phase 0**: DEV 운영 중 (`bots/invest/`)
- KIS 국내/해외: PAPER_MODE=true (실거래 전환 예정)

### 지원 명령어 (bot_commands)

| command | 설명 | 자연어 예시 |
|---------|------|------------|
| `pause_trading` | 거래 일시정지 (flag 파일 생성) | "루나 정지해", "매매 멈춰", "거래 중지", "루나 일시정지", "투자 멈춰" |
| `resume_trading` | 거래 재개 (flag 파일 삭제) | "루나 재개해", "매매 다시 시작해", "거래 재개", "루나 다시 켜" |
| `force_report` | 투자 리포트 즉시 발송 | "루나 리포트 줘", "투자 현황 보고해", "포트폴리오 보여줘", "수익률 알려줘" |
| `get_status` | 현재 상태 조회 (잔고, 모드, 사이클) | "루나 상태 어때", "투자 현황", "잔고 얼마야", "USDT 얼마 있어", "포지션 어때" |

## 🔧 클로드팀 — 소프트웨어 유지보수

**역할**: ai-agent-system 코드 점검, 보안 감사, 기술 트렌드 분석

### 팀원

| 봇 | 역할 | 실행 주기 |
|----|------|----------|
| **덱스터** | 시스템 점검 (코드 무결성·보안·DB) | 1시간 자동 (launchd) |
| **아처** | 기술 인텔리전스 (AI 트렌드 수집·분석) | 매주 월요일 09:00 KST |
| **에릭** | Explore 에이전트 (코드베이스 탐색) | 수동 |
| **케빈** | Plan 에이전트 (설계·구현 계획) | 수동 |
| **브라이언** | Bash 에이전트 (명령 실행) | 수동 |

### 지원 명령어 (bot_commands)

| command | 설명 | 자연어 예시 |
|---------|------|------------|
| `run_check` | 덱스터 기본 점검 | "덱스터 점검해", "시스템 점검해줘", "서버 괜찮아?", "점검 해봐", "보안 체크" |
| `run_full` | 덱스터 전체 점검 (npm audit 포함) | "덱스터 전체 점검", "완전 점검해줘", "풀 점검", "npm audit 해줘" |
| `run_fix` | 덱스터 자동 수정 | "덱스터 수정해줘", "자동 수정해", "패치해줘", "고쳐줘" |
| `daily_report` | 덱스터 일일 보고 (텔레그램 발송) | "덱스터 일일 보고", "일일 리포트 줘", "오늘 보고서" |
| `run_archer` | 아처 실행 (기술 트렌드 수집+분석) | "아처 실행해", "기술 소화해줘", "AI 트렌드 알려줘", "최신 LLM 소식", "아처 보고" |

## 📡 제이 (오케스트레이터) — 슬래시 명령

제이 자신이 직접 처리하는 명령:

| 슬래시 | 자연어 예시 | 설명 |
|--------|------------|------|
| `/status` | "시스템 상태", "전체 현황", "다들 살아있어?" | 전체 팀 현황 |
| `/cost` | "비용 얼마야", "토큰 얼마 썼어", "LLM 비용" | LLM 사용량·비용 |

| `/mutes` | "무음 목록", "뭐 무음했어" | 활성 무음 목록 |
| `/brief` | "브리핑", "야간 알람 뭐 있어" | 야간 보류 알람 |
| `/mute <대상> <시간>` | "루나 1시간 무음", "전체 30분 조용히" | 무음 설정 |
| `/unmute <대상>` | "루나 무음 해제", "다시 알림 켜줘" | 무음 해제 |
